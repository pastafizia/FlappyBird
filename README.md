# Flappy Bird NEAT AI

This repository contains a project developed for the Artificial Intelligence course.

The goal of the project is to apply the NEAT (NeuroEvolution of Augmenting Topologies) genetic algorithm to train an artificial agent capable of playing Flappy Bird.

## Project Overview

The project focuses on using neuroevolution to evolve neural networks that control a Flappy Bird agent.
Through generations of evolution, the algorithm improves the behavior of the agents based on their performance in the game environment.

The implementation demonstrates how evolutionary algorithms can be used to automatically develop neural network structures and weights.

## Flappy Bird Environment

The Flappy Bird game environment used in this project was not originally developed by me.
It is used as a test environment where the AI agents interact with the game and learn through evolutionary processes.

## Objective

The objective of the project is to explore evolutionary machine learning techniques and observe how the NEAT algorithm can evolve increasingly effective agents over multiple generations.

## Notes

This project was developed for academic purposes as part of an Artificial Intelligence course.
